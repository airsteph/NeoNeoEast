# NeoNeoEast · 外语学习助手

一个零配置的 Chrome 扩展:在你正常浏览中文网页时,悄悄把**少量**中文词替换成你正在学的外语(英语 **或** 日语),鼠标悬停即可看到释义、音标/词性和例句,并配有一套轻量的间隔复习循环。灵感来自 Toucan,界面采用浅色印刷手册质感。

> 装上就能用,不需要任何账号、不需要联网翻译、不上传任何数据——所有词库、例句与学习记录只存在你自己的浏览器里。

## ✨ 功能特点

- **沉浸式替换**:阅读中文网页时,按设定的间隔把认识的中文词替换成外语,自然地在语境中记单词。
- **中文分词精准切词**:使用浏览器内置的 `Intl.Segmenter` 按真实词边界切分,不会切错词。
- **英 / 日双语,互斥切换**:同一时间只学一种语言,专注不分心。
  - 英语:高考 / 四级 / 六级 / 考研 / 托福 / 雅思 / GRE
  - 日语:JLPT N5 / N4 / N3 / N2 / N1
- **分层词库,层层递进**:按考试等级多选,只替换你选中的层级、且尚未掌握的词。
- **悬停卡片**:释义、音标(英语)/ 词性(日语)、例句(母语例句 + 中文翻译)、以及「无须学习 / 记录生词」快捷按钮。
- **间隔复习**:标记「记录生词」的词会进入复习队列;弹窗内可做每日复习,含每日上限与顺延机制。
- **出词密度可调**:滑块控制"每屏大约出现几个词",按词间隔均匀铺满全文,不扎堆。
- **黑名单**:在指定网站上完全不替换。
- **导出 / 导入**:CSV 导出(全部 / 已掌握 / 在学 / 待复习),以及 `.json` 完整备份,换电脑一键恢复。

## 📦 安装(开发者模式加载)

1. 下载本仓库(点右上角绿色 **Code → Download ZIP**),解压。
2. 打开 Chrome,地址栏输入 `chrome://extensions` 回车。
3. 打开右上角的 **开发者模式**。
4. 点 **加载已解压的扩展程序**,选择解压出来的文件夹。
5. 打开任意一篇长中文文章,刷新页面即可看到效果。点击工具栏的扩展图标可调整语言、层级和密度。

> 需要 Chrome 87+(依赖 `Intl.Segmenter`)。修改设置后,刷新对应网页生效。

## 🗂️ 项目结构

```
NeoNeoEast/
├── manifest.json            # 扩展清单(Manifest V3)
├── background/              # 后台服务 worker:学习状态机、复习调度、统计
├── content-script/          # 注入网页的替换逻辑 + 悬停卡片样式
├── popup/                   # 工具栏弹窗:开关、语言/层级/密度、复习卡片、7 日趋势
├── options/                 # 设置页:黑名单、复习节奏、导出/导入、数据概览
├── data/                    # 分层词库(dict-en.json / dict-ja.json)
└── icons/                   # 图标(16/32/48/128)
```

## 🔒 隐私

本扩展**不联网、不发送任何请求**。词库、例句与中译均随扩展本地打包,所有学习记录保存在浏览器的 `chrome.storage.local` 中。卸载即清除。MIT 开源许可。

部分日语例句来自 Tatoeba 项目(CC-BY 2.0),详见 `data/SOURCES.md`。

## 🛠️ 技术栈

纯原生 JavaScript / HTML / CSS,零构建、零依赖、零联网。Manifest V3 标准,`Intl.Segmenter` 做中文分词,`chrome.storage.local` 存词库与学习数据。
